# **개인스터디용 게시판 API 서버 개발**

## 📝 목표
- 계층별 단위테스트 작성 능력 
- 가독성있고 유지보수성을 고려하여 최대한 객체지향적으로 설계
- 1일 1커밋 시도
- 학습했던 개발지식을 내것으로 만들기
- 추후 다른 스킬스택 학습을 위해 새로운 서비스 추가

## 📈 기술 스택
- JAVA 8<br>
- Spring Boot<br>
- MySQL<br>
- Spring Data Jpa<br>
- Querydsl<br>
- JUnit5<br>
- Mocktito
